# youtube_api API Documentation

**Base URL**: https://www.googleapis.com/youtube/v3
**Auth Type**: api_key
**Description**: YouTube API allows for managing videos, playlists, channels, and retrieving YouTube data like video search and playback.

---

**YouTube Data API**

- **Base URL**: https://www.googleapis.com/youtube/v3

### Authentication:
- Each request requires an API key or OAuth 2.0 token.

### Endpoints:
- **Videos**: Manage and retrieve videos.
- **Playlists**: Create and manage playlists.
- **Channels**: Retrieve and manage channel information.
- **Search**: Search YouTube contents including videos, playlists, and channels.

### Usage:
- Use standard HTTP methods (`GET`, `POST`, `PUT`, `DELETE`) to interact with resources. Operations that modify data or require access to private user information require OAuth tokens.

### Additional Tools:
- **YouTube Analytics API**: Metrics and analytics for videos and channels.
- **YouTube Live Streaming API**: Manage live streaming.
